# David Wong — iOS Portfolio

A static, GitHub Pages-ready portfolio for five iOS product concepts built with ChatGPT:

- Deal Scanner
- CraveCompass
- Anti-Social Network
- ScreenShotScan
- Wild Atlas

The website uses real iOS Simulator recordings, a continuous vertical playlist, a draggable project scrubber, and keyboard/touch navigation. The original iOS source projects are not included in this repository.

GitHub Pages serves the `docs/` directory. Open `docs/index.html` directly or run any local static server from `docs/` to preview the site.
